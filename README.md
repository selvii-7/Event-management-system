# Event Management System

## Overview
The Event Management System is a web-based application developed using Java, JSP, Servlets, HTML, CSS, and MySQL. The system helps users manage events, register accounts, log in securely, and maintain event-related information.

## Features
- User Registration and Login
- Event Creation and Management
- Budget Management
- Database Connectivity
- User-Friendly Interface

## Technologies Used
- Java
- JSP
- Servlets
- HTML
- CSS
- MySQL
- NetBeans IDE

## Purpose
This project was developed as an academic project to demonstrate web application development using Java technologies and database connectivity.

## Project Modules
- Registration Module
- Login Module
- Event Management Module
- Budget Management Module
